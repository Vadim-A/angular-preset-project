Пример использования i18n

В i18n можно передавать дополнительное описание в формате "Значение|Описание" или "Описание" Id переводимого блока генерируется на основе хэша текста внутри тэга

По умолчанию i18n генерирует для блока токен на основе хэша содержимого переводимого блока. Но можно указать свой токен, используя @@.
